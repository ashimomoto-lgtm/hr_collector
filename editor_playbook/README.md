# 1on1総研 編集長プレイブック

- マガジン: https://note.com/kajiken0630/m/m1586f35de51a
- 取得API: https://note.com/api/v1/magazines/m1586f35de51a/notes?page=1

## ファイル構成

| ファイル | 用途 |
|---|---|
| `principles.md` | 全38記事を貫く **10の編集原則**。判断軸として常時参照 |
| `frameworks.md` | 個別記事の **方法論辞典**。企画・分析・意思決定の引き出し |
| `checklists.md` | **企画前 / 公開前 / 月次振り返り** の運用チェックリスト |
| `prompts.md` | Claude/Gemini に貼って使う **編集長プロンプト集**（企画ブレスト、リライト判定、UVP抽出など） |

## 使い方

### 日常の編集判断
「この企画は『器の中の整理』か『器の外』か？」「この記事のUVPを20字で書ける？」など、`principles.md` の10原則をチェックしながら進める。

### 月次の運用
`checklists.md` の「月次振り返り」を編集会議のアジェンダにする。Profit Tree の更新、PV TOP/BOTTOM 5記事の棚卸し、引き算判断（削除/統合/リライト）の3点が核。

### 新企画のブレスト
`prompts.md` のテンプレを Claude に投げ、UVP を 20 字で出させる→ Goal Oriented で四半期テーマと紐づくか確認 → カテゴリ啓蒙 vs 自社宣伝の比率を出す。
