## A. ブランディングコンテンツ（第一想起獲得・専門性訴求）

### A-1. AI時代のマネジメントは「対話」に回帰する—生成AIがマネジャーの役割をどう変えるか

| 項目 | 内容 |
|------|------|
| **想定読者** | 経営層、人事担当者、マネジャー |
| **切り口** | 生成AIがタスク効率化を進める一方で、マネジャーの対話スキルや人間的なリーダーシップの重要性が増す。AIでは代替できない「人」に焦点を当てたマネジメントの再定義を論じます。 |
| **使用するネタ** | 生成AIを仕事で使っても、なぜ私たちは忙しいままなのか / AI時代のマネジメント業務と学習環境に関する実態調査 / AI-Readyな企業の条件とは |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | AI技術の急速な進化と導入が進む中で、その真の価値を引き出すマネジメントのあり方が問われているため。 |

### A-2. 曖昧な人事評価が信頼を壊す—1on1で「納得感」を高める評価フィードバックの鍵

| 項目 | 内容 |
|------|------|
| **想定読者** | 人事担当者、マネジャー・管理職 |
| **切り口** | 人事評価の基準が曖昧だと従業員のストレスを高め、組織への信頼を損ないます。評価結果を一方的に伝えるのではなく、1on1を通じて対話し、納得感を醸成するマネジャーの役割と具体的なアプローチを深掘りします。 |
| **使用するネタ** | 組織における評価制度の運用状況と社員心理への影響を調査 / 賃上げ局面で顕在化する報酬設計の盲点 |
| **接続する資料** | 上司・部下の信頼関係16段階シート |
| **Why Now** | 賃上げや報酬制度の見直しが進む中で、評価の透明性と納得感が従業員のエンゲージメントに直結するため。 |
