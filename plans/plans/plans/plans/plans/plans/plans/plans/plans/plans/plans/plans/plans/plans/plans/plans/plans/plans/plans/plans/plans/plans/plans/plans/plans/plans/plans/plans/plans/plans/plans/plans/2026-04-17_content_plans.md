## A. ブランディングコンテンツ（第一想起獲得・専門性訴求）

### A-1. AI時代に問われるマネジャーの「人間力」—生成AIは上司と部下の関係をどう変えるか

| 項目 | 内容 |
|------|------|
| **想定読者** | マネジャー・管理職、経営層、人事担当者 |
| **切り口** | 生成AIの普及が部下のキャリア観や上司への信頼に影響を与えている現状を踏まえ、AIでは代替できないマネジャーの役割と、人間的な対話の重要性を考察します。特に「上司より生成AIを信頼する」という若手の声から、マネジャーが向き合うべき課題を深掘りします。 |
| **使用するネタ** | 若手がキャリア相談でAIを上司より信頼する調査結果 / 課長のAI活用実態調査 / AI活用がキャリアに与える影響に関する分析 |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | 生成AIの進化が止まらない中、マネジメントの現場で具体的にどのような変化が起きているかを理解し、対応策を考える必要があるためです。 |

### A-2. 「スター頼み」から脱却！高業績チームを育むマネジメントの10の視点

| 項目 | 内容 |
|------|------|
| **想定読者** | 経営層、人事担当者、マネジャー・管理職 |
| **切り口** | 個々の優秀な人材や偶発的な成功に頼るのではなく、組織として高業績チームを持続的に生み出すための構造的なアプローチを提示します。チームの機能不全を引き起こす課題を掘り下げ、人事・マネジメントが取り組むべき本質的な視点を解説します。 |
| **使用するネタ** | 高業績チームを生むための10の視点に関する記事 / 1on1が機能しない原因の分析 / 風通しの良い組織を作るためのポイント |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | 多くの企業がチームの成果向上に課題を抱え、その解決策として個人の能力だけでなく組織全体の機能に注目が集まっているためです。 |

### A-3. キャリア自律を「設計」する時代へ—イケア・ジャパンに学ぶ、選び選ばれる組織の条件

| 項目 | 内容 |
|------|------|
| **想定読者** | 人事担当者、経営層、組織開発担当者 |
| **切り口** | キャリア自律は個人の「やる気」だけでなく、企業が制度や文化で「設計」することで誰もが動き出せるという視点を提供します。社内公募制度を成功させるためのポイントや、従業員が自律的にキャリアを形成し、企業に選ばれ続けるための組織のあり方を考察します。 |
| **使用するネタ** | イケア・ジャパンの社内公募制度導入事例 / キャリア自律を設計で考えるという書籍紹介 / AIを活用したキャリア支援の重要性 |
| **接続する資料** | 0から始める！1on1導入ガイドライン |
| **Why Now** | 労働市場の流動性が高まる中、企業が従業員のキャリア自律を支援し、定着と成長を両立させるための具体的な戦略が求められているためです。 |

### A-4. 人的資本経営時代の「給与の未来展望」—賃上げと納得感を両立する報酬設計の哲学

| 項目 | 内容 |
|------|------|
| **想定読者** | 経営層、人事担当者、HRBP |
| **切り口** | インフレと人材不足が続く現代において、単なる賃上げに留まらない、従業員の「納得感」と「未来への展望」を育む報酬設計の重要性を論じます。報酬が離職や育成停滞を招かないよう、企業が示すべきメッセージと、従業員の「働く合理性」に訴えかける人事制度のあり方を考察します。 |
| **使用するネタ** | インフレ・人材不足時代の給与に関する調査 / 賃上げ局面で顕在化する報酬設計の課題 / ベースアップに関する意識調査 |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | 物価上昇と賃上げが社会的な関心事となる中で、報酬制度が従業員のエンゲージメントと定着に与える影響を深く理解し、戦略的な人事施策を講じる必要性が高まっているためです。 |
