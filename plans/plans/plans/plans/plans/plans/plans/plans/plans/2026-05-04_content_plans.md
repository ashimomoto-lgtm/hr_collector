## A. ブランディングコンテンツ（第一想起獲得・専門性訴求）

### A-1. AI時代にマネジャーの「人間力」が問われる：1on1で育む部下の自律と創造性

| 項目 | 内容 |
|------|------|
| **想定読者** | マネジャー・管理職、人事担当者（HRBP含む） |
| **切り口** | AIが業務効率化を進める中で、マネジャーは「人の育成」「対話」「キャリア支援」といった人間ならではの役割に注力する必要があります。AIによって顕在化するマネジメントの新たな課題と、1on1が果たすべき役割を考察します。 |
| **使用するネタ** | 企業のAI活用と人材戦略調査 / AI時代のマネジメント業務と学習環境調査 / 人事×AIの真価は「データ基盤」で決まる / 生成AI活用後の業務効率化課題 |
| **接続する資料** | マネジャーのための1on1完全ガイド、1on1総研 調査レポート（各種） |
| **Why Now** | AIの進化が急速に進み、企業が具体的な活用と人材戦略を模索しているため。 |

### A-2. 「静かな退職」と「クォーターライフクライシス」：若手層のリアルとマネジメントの役割

| 項目 | 内容 |
|------|------|
| **想定読者** | マネジャー・管理職、人事担当者（HRBP含む） |
| **切り口** | 若手層に広がる「静かな退職」や「クォーターライフクライシス」の実態を分析し、その背景にある価値観や不安を深掘りします。これらの課題に対して、マネジャーが1on1を通じてどのように部下の本音を引き出し、エンゲージメントを高め、キャリアを支援すべきかを考察します。 |
| **使用するネタ** | 正社員の静かな退職に関する調査 / 正社員のクォーターライフクライシス調査 / 職場の孤独と経営課題 |
| **接続する資料** | 上司・部下の信頼関係16段階シート、メンバーのための1on1完全ガイド |
| **Why Now** | 若手層の離職やエンゲージメント低下が企業の喫緊の課題となっているため。 |

### A-3. 「評価納得度」と「賃上げ期待」のギャップを埋める：1on1が果たす報酬対話の役割

| 項目 | 内容 |
|------|------|
| **想定読者** | 人事担当者（HRBP含む）、マネジャー・管理職 |
| **切り口** | 評価基準の曖昧さがストレスや納得度の低下を招き、賃上げも「限界認識」を生む現状を指摘します。金銭的報酬だけでなく、評価やキャリアに関する「納得と説明」が重要であることを強調し、1on1が報酬対話の場として、いかに社員の納得感を高め、モチベーションを維持・向上させるかを論じます。 |
| **使用するネタ** | 評価制度運用と社員心理の調査 / ベースアップに関する意識調査 / 賃金改定と報酬制度の見直し |
| **接続する資料** | マネジャーのための1on1完全ガイド、マネジャー向け1on1質問集 |
| **Why Now** | 賃上げが企業の重要課題となる中で、社員の納得感を高める報酬制度運用が求められているため。 |

### A-4. 「対話のすれ違い」を乗り越える：信頼関係を築く1on1の真髄

| 項目 | 内容 |
|------|------|
| **想定読者** | マネジャー・管理職、人事担当者（HRBP含む） |
| **切り口** | 職場における対話の「すれ違い」の根本原因を深掘りし、それが組織の生産性やエンゲージメントに与える悪影響を解説します。心理的安全性を確保し、本音で語り合える「建設的な対話」を育むための1on1の設計と、マネジャーに求められるスキルを具体的に提案します。 |
| **使用するネタ** | 職場の対話のすれ違いに関する書籍紹介 / 価値観の多様化とコミュニケーション課題 / 風通しの良い組織作り |
| **接続する資料** | 上司・部下の信頼関係16段階シート、マネジャーのための1on1完全ガイド |
| **Why Now** | 価値観の多様化が進み、職場内のコミュニケーションギャップが顕在化しているため。 |

### A-5. スキルベース組織への移行期：マネジャーに求められるキャリア支援と1on1

| 項目 | 内容 |
|------|------|
| **想定読者** | 人事担当者（HRBP含む）、経営層、マネジャー・管理職 |
| **切り口** | リスキリングやスキルベース組織への注目が高まる中で、多くの企業がスキル管理の「失敗ループ」に陥っている現状を打破するための戦略を提示します。マネジャーが1on1を通じて、部下のスキルを可視化し、キャリア自律を支援し、組織の変革に貢献する方法を考察します。 |
| **使用するネタ** | スキル管理の失敗ループと改善点 / スキルベース組織に関する書籍紹介 / ポストAI時代の育成・キャリアリスク |
| **接続する資料** | メンバーのための1on1完全ガイド、マネジャーのための1on1完全ガイド |
| **Why Now** | AI時代における個人のスキル変革と、それを支える組織の人材マネジメントが急務となっているため。 |
