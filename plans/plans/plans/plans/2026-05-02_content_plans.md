## A. ブランディングコンテンツ（第一想起獲得・専門性訴求）

### A-1. AI時代のマネジメント変革：人間中心の対話が組織の未来を拓く

| 項目 | 内容 |
|------|------|
| **想定読者** | 経営層、人事担当者（HRBP含む） |
| **切り口** | AIの進化が業務効率化だけでなく、人の役割やマネジメントのあり方そのものを変革している現状をデータで示します。その上で、AIでは代替できない人間中心の対話と信頼関係構築の重要性を提唱します。 |
| **使用するネタ** | AI時代の転職・採用調査 / AI活用が変える職場とマネジメント調査 / AIと人間対話の重要性 |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | AI技術が急速に進化し、企業での導入が進む中で、マネジメントの再定義が急務となっているためです。 |

### A-2. 「静かな退職」の深層：日本企業が直面するエンゲージメントの危機をデータで読み解く

| 項目 | 内容 |
|------|------|
| **想定読者** | 経営層、人事担当者、マネジャー・管理職 |
| **切り口** | 正社員の4割以上が「静かな退職」をしているというデータから、その背景にある従業員心理、組織文化、マネジメントの課題を深掘りします。KAKEAI独自の視点で本質的な解決策を提示し、組織の活性化を促します。 |
| **使用するネタ** | 正社員の静かな退職に関する調査 |
| **接続する資料** | 1on1総研 調査レポート（各種） |
| **Why Now** | 「静かな退職」が社会現象化し、企業の人材定着における喫緊の課題となっているためです。 |

### B-2. 「静かな退職」予備軍を見逃さない！部下の本音を引き出す1on1対話術

| 項目 | 内容 |
|------|------|
| **想定読者** | マネジャー・管理職 |
| **切り口** | 「静かな退職」を防ぐために、マネジャーが1on1で部下の不満や不安、キャリア志向を早期に察知し、信頼関係を築きながら本音を引き出すための具体的な質問や傾聴のポイントを解説します。 |
| **使用するネタ** | 正社員の静かな退職に関する調査 |

| **Why Now** | マネジャーが「静かな退職」の兆候を早期に捉え、適切な対応を取ることが組織の活性化と人材定着に直結するためです。 |
