2 HR界隈トレンド分析

1on1・マネジメント領域

1on1は機能不全に陥るケースがあるものの、「パーソル総合研究所」のコラム「なぜ1on1は機能しないのか？」「1on1という「旅」を始める」が示すように、その重要性は依然として高く、対話の再設計が求められています。上司と部下のキャリア面談の効果向上や、「ダイヤモンド・オンライン」の「仕事ができる上司の当たり前」に見られるような上司のコミュニケーション能力向上が課題です。また、「パーソル総合研究所」のコラムでは、フォロワーシップの育成や、多様性を活かすためのリーダー行動のアップデート、建設的なコンフリクトの醸成が、組織の成長に不可欠であると指摘されています。さらに、「日本の人事部」の書籍紹介「最強の組織」や「大課長問題」からも、マネジメントの質が組織の成果を大きく左右する動向が見て取れます。

人事制度・採用・育成

人事制度・採用・育成領域では、企業が持続的な成長のために制度の刷新や多様な人材確保に注力する動向が見られます。「カルビー」や「レオパレス21」のように人事制度を抜本的に改定する動きがあり、ジョブ型人材マネジメントへの関心も高まっています。採用においては、「ニトリ」が通年採用を導入し、新卒採用市場では「マイナビ調べ」の「就職企業人気ランキング」や「ワークス大卒求人倍率調査」が活発な動きを示しています。また、デロイト調査で人材流出・獲得困難が最優先リスクとされており、デジタルスキル標準の公表や「Schoo for Business」による育成計画パッケージ提供など、従業員のスキルアップとキャリア自律支援が重要視されています。「静かな退職」や「ホワイトハラスメント」といった従業員のエンゲージメントに関わる課題にも企業は向き合っており、賃上げや副業の拡大といった働き方の多様化への対応も進んでいます。

HRテック・AI活用

HRテック・AI活用領域では、AIが人事の様々な業務に深く浸透し、その活用範囲が拡大しています。「日本の人事部」の「AI時代のマネジメント業務と学習環境に関する実態調査」や「HERP」の「AI活用の実態調査」が示すように、AIの導入は進む一方で、人が関わることへの意識も依然として高いです。採用領域では、「talentbook」の「タレントブック・キャンパス」や「マイナビ」の「WHAT」がAIを活用した面談コーディネートやAI面接を提供し、「ギブリー」はAI面接のインタラクティブ機能を強化しています。また、「ビズリーチ」の「Onboard AI」によるオンボーディング支援や、「On Technologies」のAIエージェント「WorkOn」など、多様なHRテックサービスが登場。タレントマネジメントシステムや統合型人事システムの導入も進んでおり、「HRBrain」や「ジンジャー」などの事例が見られます。AIの真価を引き出すためには、「HRzine」や「三菱UFJリサーチ&コンサルティング」が指摘するように、AI-Readyなデータ基盤とガバナンスの構築が不可欠であり、生成AIの職場での活用実態や課題に関する調査も進んでいます。
