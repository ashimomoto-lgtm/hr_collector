2 HR界隈トレンド分析

1on1・マネジメント領域

現代のマネジメント領域では、多様な人材と働く環境の変化に対応するリーダーシップが求められています。パーソル総合研究所のコラム「1on1という『旅』を始める」が示すように、1on1は依然として重要視されつつも、その効果的な実践が課題です。ダイヤモンド・オンラインが指摘する「部下は『正しい指示』ではなく、『わかってくれる上司』に動かされる」という視点や、プレジデントオンラインが報じる「管理職を拒否するイマドキの若手社員」の動向を受け、上司にはより共感的な姿勢が求められています。また、パーソル総合研究所の「個人の人的資本を高めるだけでは、組織の成果につながらない？」が示すように、人的資本経営においては「対話」を通じた組織全体のエンゲージメント向上が鍵とされており、ジョブ型人材マネジメント（NECビジネスインテリジェンスの事例）や多様性を活かすリーダー行動のアップデートも重要な動向です。

人事制度・採用・育成

人事制度・採用・育成領域では、多様な働き方とキャリア自律を支援する動きが顕著です。カルビーが約17年ぶりに人事制度を抜本的に刷新し、多様な貢献と成長を促す新制度を導入したように、企業は柔軟な制度設計を進めています。採用においては、東洋経済オンラインが報じる「超スピード退職」問題や、就活会議が導入した生成AIを悪用した虚偽投稿への対策など、学生の行動変化に対応した新たな課題が浮上しています。育成面では、AI時代を見据え、ダイヤモンド・オンラインが提唱する組織の"縦軸"と"横軸"での再設計による人材育成が不可欠とされています。また、パーソル総合研究所の「日本の『離職理由』はどう変化したのか」が示すように、リテンション・マネジメントの重要性が増しており、キャリア面談の強化や副業の容認など、個人のキャリア形成を支援する施策が注目されています。

HRテック・AI活用

HRテック・AI活用の領域では、生成AIの導入が急速に進み、人事プロセスの効率化と高度化が図られています。三井住友信託銀行が「AI面接官」を導入し公平・多様な採用プロセスを実現したように、採用活動におけるAI活用が広がっています。また、HRBrainはAI新機能「HRBrain Brain」でパーソナライズ回答や高度な人事データ分析を提供し、SmartHRはANAグループ29社に導入され業務効率化とタレントマネジメントを推進するなど、HRテックベンダーによる機能強化が活発です。一方で、日本の人事部による「課長のAI活用実態調査」やパーソル総合研究所の「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」といった調査からは、現場での活用実態や課題も浮き彫りになっており、単なる導入だけでなく、効果的な活用を促すための施策が求められています。
