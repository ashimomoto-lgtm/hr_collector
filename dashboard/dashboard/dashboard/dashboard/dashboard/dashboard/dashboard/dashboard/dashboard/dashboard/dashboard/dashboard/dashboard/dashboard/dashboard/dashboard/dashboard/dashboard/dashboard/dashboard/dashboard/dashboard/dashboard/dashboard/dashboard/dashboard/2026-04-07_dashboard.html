1on1・マネジメント領域

1on1の重要性は再認識されつつも、その効果的な実施方法や上司のコミュニケーションスキルが課題となっています。「パーソル総合研究所」のコラム「1on1という『旅』を始める」では、継続的な取り組みの必要性が示される一方、「ダイヤモンド・オンライン」の記事「1on1で部下がスッとシャッターを下ろす上司の"痛い話し方"」では、対話の質が重要であることが浮き彫りになっています。また、多様な人材を活かすためのリーダーシップ行動のアップデートが求められ、「パーソル総合研究所」の「多様性を活かすためのリーダー行動アップデート」がその一例です。パワハラ防止やシニア世代の部下への支援など、マネジメント層にはより複雑な対人スキルと組織改革への視点が求められています。

人事制度・採用・育成

人的資本経営の推進が求められる中で、個人のキャリア自律支援と組織成果への接続が重要視されています。「パーソル総合研究所」の「個人の人的資本を高めるだけでは、組織の成果につながらない？」では対話を通じた連携の必要性が説かれ、「カルビー」のように約17年ぶりに人事制度を抜本的に刷新し、多様な貢献と成長を促す動きも見られます。採用においては、AI時代における学生の「見極め」の難しさから「観察力」と「対話力」が重視される傾向にあり（「日経ビジネス」の「学生の『見極め』は諦めよ」）、中途採用・転職市場も活発で、特に20代若手社員のスピード退職や転職動向が注目されています。

HRテック・AI活用

HRテック領域では、AI、特に生成AIの活用が急速に進んでいます。採用活動では「三井住友信託銀行」が「AI面接官」を導入し公平・多様なプロセスを実現する一方、「日本の人事部」の調査では26卒・27卒学生の就活におけるAI活用実態が明らかになっています。人事・労務業務の効率化も進んでおり、「SmartHR」のANAグループ29社への導入や、「HRBrain」がAI新機能「HRBrain Brain」を提供するなど、タレントマネジメントシステムとAIの連携が強化されています。さらに、「マネーフォワード AI Cowork」や「キャスター」の「AI社員」のように、AIがバックオフィス業務を自律的に遂行するAIエージェントサービスの登場が注目されていますが、「パーソル総合研究所」のコラム「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」が示すように、AI導入だけでは根本的な業務課題解決には至らない可能性も指摘されています。
