2 HR界隈トレンド分析

1on1・マネジメント領域

マネジメント領域では、1on1の効果的な運用と上司の質が改めて問われています。パーソル総合研究所のコラム「なぜ1on1は機能しないのか？」が示すように、部下の自律的な思考を促す対話の再設計が重要視されています。また、ダイヤモンド・オンラインの「部下を『モンスター社員』に変えてしまう上司の〈4つのNG行動〉」では、上司の不適切な行動が組織に与える悪影響が指摘されており、上司自身の行動変容が求められています。多様な人材を活かすためのリーダー行動のアップデートや、建設的なコンフリクトを育む組織づくりの重要性もパーソル総合研究所のコラムで語られており、経営リーダーの育成にも注目が集まっています。

人事制度・採用・育成

人事制度・採用・育成領域では、人的資本経営の視点から採用・育成戦略の変革が加速しています。リクルートMSの「企業の新卒採用実態調査2026」によると、多くの企業が採用手法の変革を実感しており、日立が実践する「タレントプール」やJ:COMの「アルムナイ採用制度」新設など、多様な人材獲得チャネルが模索されています。カルビーが約17年ぶりに人事制度を刷新したように、既存の制度を見直し、社員のキャリア自律を促す動きも活発です。また、ミツカリがカオナビグループ入りしたように、人的資本データの蓄積・活用による戦略的な人事が重視され、地方創生における人材誘致の取り組みも注目されています。

HRテック・AI活用

HRテック領域では、AI、特に生成AIの活用が急速に進展しています。イオングループが「PeopleX AI面接」を導入し、パーソルHDが管理職の目標設定支援AIエージェントを開発するなど、採用からマネジメントまで幅広い業務でAIの導入が進んでいます。すかいらーくレストランツが現場向けAIエージェント「ABILI Pal」を2600店舗に導入した事例のように、現場業務の効率化にもAIが活用されています。また、「26卒・27卒【就活・AI活用実態調査】」や「大学生の就活におけるAI活用実態調査」が示すように、就職活動におけるAIの利用も一般化し、企業側も入社時研修で生成AIを必修化するなど、AIリテラシーの向上が重視されています。
