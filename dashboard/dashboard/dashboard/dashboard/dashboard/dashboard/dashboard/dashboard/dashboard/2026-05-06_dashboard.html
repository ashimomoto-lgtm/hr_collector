1on1・マネジメント領域

1on1や対話の質向上への関心が高く、「なぜ1on1は機能しないのか？」（パーソル総合研究所）や「なぜ対話はすれ違うのか？」（HRzine）といった記事に見られるように、その効果的な運用が課題となっています。AI活用は「管理職罰ゲーム」解消から（日経ビジネス）と指摘されるように、AI時代のマネジメント業務の見直しが急務です。多様な働き方やキャリア自律を支援するため、上司と部下のキャリア面談の質向上や、フォロワーシップ、経営リーダー育成の重要性が増しています。「NTTドコモ」のキャリア開発施策のように、個人の成長と組織の成長を両立させる取り組みが注目されています。

人事制度・採用・育成

採用市場ではAI活用が進む一方で、「AI時代の転職・採用に関するアンケート調査」（日本の人事部）が示すように、人が関わることへの意識は依然として高いです。「カルビー」が約17年ぶりに人事制度を刷新したように、企業は人的資本経営を意識した制度改革を推進しており、スキル管理の重要性も高まっています。Z世代の仕事観やクォーターライフクライシスといった従業員の意識変化に対応するため、育児・介護休業支援制度（国際航業の「サンクス・シェア」）や多様な働き方への対応が求められています。新卒採用での生成AI必修化（日本の人事部）など、AI時代に対応した育成が始まっています。「正社員の静かな退職に関する調査」（日本の人事部）から、従業員エンゲージメントやキャリア自律支援の重要性が浮き彫りになっています。

HRテック・AI活用

HR領域におけるAI活用は、採用面接（JetBの「Our AI面接」）やOB/OG訪問（talentbookの「タレントブック・キャンパス」）など、具体的な業務支援に広がっています。タレントマネジメントシステム（「カオナビ」の京王電鉄導入、「HRBrain」の一橋大学導入）や人事評価システム（「freee人事評価」）の導入が進み、人事情報の統合管理による効率化が図られています。AI活用は個人の業務効率化だけでなく、「リクルートMS調べ」が示すように組織全体の生産性向上へと広がる段階にあり、AIエージェント構築・活用人材の育成支援（ギブリーの「Claude Cowork 活用支援」）も登場しています。しかし、「生成AIはなぜ職場に広がらないのか」（パーソル総合研究所）といった調査から、その普及にはまだ課題があることも示唆されています。「米オラクル」が人事業務向けの「Fusion Agentic Applications」を発表するなど、HRテックベンダーはAIを活用したより高度なソリューション提供を強化しています。
