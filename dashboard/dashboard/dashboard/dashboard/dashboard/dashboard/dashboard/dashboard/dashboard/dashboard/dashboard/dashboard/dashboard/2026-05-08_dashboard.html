2 HR界隈トレンド分析

1on1・マネジメント領域

1on1に関しては、「なぜ対話はすれ違うのか？」といった書籍紹介や、パーソル総合研究所のコラム「なぜ1on1は機能しないのか？」が示すように、その効果的な運用方法や再設計が喫緊の課題と認識されています。AI時代の到来により、マネジメント業務も変革が求められており、「AI時代のマネジメント業務と学習環境に関する実態調査」（日本の人事部）などが実施され、上司の役割への影響が注目されています。「いい上司」になろうとして苦しくなる状況（ダイヤモンド・オンライン）や、部下のキャリア自律を促す面談の重要性が指摘されており、リーダーシップのあり方が問われています。「社員の声と組織に関する意識」調査（日本の人事部）や「人的資本経営に関する意識調査」が示すように、従業員のエンゲージメントや組織文化への関心が高まっています。「フォロワーシップ」の育成や「健全な対立」を育む組織作りなど、多様性を活かすためのリーダーシップのアップデートが求められています。

人事制度・採用・育成

人事制度においては、カルビーが約17年ぶりに制度を刷新したように、企業は変革期にあり、「ジョブ型人事を導入すれば解決する」という安易な考え方への警鐘も鳴らされています。ダイバーシティ経営の推進が重要視され、「女性が活躍する会社BEST100」の発表や、国際航業による育児・介護休業を支える「サンクス・シェア」制度導入事例が見られます。採用活動では、2027年卒の「ワークス大卒求人倍率調査」が発表される一方、「AI時代の転職・採用に関するアンケート調査」（日本の人事部）が示すように、AIが採用プロセスに与える影響が注目されています。「正社員のクォーターライフクライシス調査2026年」や「Z世代の仕事観と自分らしさに関する調査2026」が示すように、多様な働き方やキャリア観への対応が企業の課題となっています。スキル管理の失敗ループから抜け出すための議論や、NTTドコモのキャリア開発施策のように、社員のキャリア自律を促す育成が重視されています。

HRテック・AI活用

HRテック領域では、AIの活用が急速に進んでおり、川崎重工業がAIキャリアエージェント「BaseMe」を導入したり、Insight Edgeが生成AIを活用したスキル評価システムを開発するなど、具体的な導入事例が増加しています。採用プロセスにおけるAI活用も顕著で、talentbookがAI時代のOB/OG訪問を実現するサービスを開始し、PeopleXやJetBはAI面接サービスで企業と候補者の適合度合いを自動判定する機能を提供しています。一方で、AI活用に関する課題も浮上しており、「AIを使って仕事をする部下」への上司の対応や、「生成AIはなぜ職場に広がらないのか」（パーソル総合研究所）といった調査が実施されています。「AI-Readyな企業の条件」（三菱UFJリサーチ&コンサルティング）が示唆するように、AIを効果的に活用するためには、メタデータとガバナンスの整備が不可欠です。MIXIが「帳票DX for SmartHR」を導入して就労証明書発行業務を自動化したように、HR業務の効率化・DX推進も引き続き重要なテーマです。
