1on1・マネジメント領域

1on1は機能不全に陥りやすいとされ、「なぜ1on1は機能しないのか？」といった問いかけから対話の再設計が求められています。上司と部下のキャリア面談の効果を高めることや、心理的安全性を損なわないためのリーダーシップが重要視されており、「真のリーダーになるための課長塾」では共感力の欠如が指摘されています。また、「建設的なコンフリクトをどう育てるか」といった多様性を活かすためのリーダー行動のアップデートが課題となっています。「サーベイや行動データは「事実」ではなく「兆し」として見よ」とHRzineが提言するように、データ活用における誤判断を避ける視点も重要です。

人事制度・採用・育成

人材流出や人材獲得の困難が4年連続で最優先リスク（デロイト調べ）と認識され、人事制度の抜本的刷新が「レオパレス21」や「カルビー」で実施されています。採用活動では「カジュアル面談に関する実態調査」が行われ、マイナビによる「27卒の「就職企業人気ランキング」」が注目を集めています。育成面では「Schoo for Business」が階層別の年間育成計画パッケージを提供し、ウィングアーク1stは次世代経営人材育成プログラムを本格展開しています。ポストAI時代を見据えた育成とキャリアの再設計が求められ、「ジョブ型人材マネジメントの「次の一手」を考える」といった動向も見られます。

HRテック・AI活用

HRテックの導入が活発化しており、兵庫県がタレントマネジメントシステム「タレントパレット」を、湖池屋が統合型人事システム「ジンジャー」を導入しています。AI活用は多岐にわたり、ビズリーチはオンボーディング支援サービス「Onboard AI」を提供し、パーソルHDは管理職の負担軽減のため社員の目標設定を支援するAIエージェントを開発しました。採用活動では「26卒・27卒【就活・AI活用実態調査】」が行われ、企業は入社時研修で"生成AI"を必修化するなど、AIリテラシー向上に注力しています。しかし、「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」といった課題も指摘され、利用率だけでは測れない職場での活用実態が模索されています。
