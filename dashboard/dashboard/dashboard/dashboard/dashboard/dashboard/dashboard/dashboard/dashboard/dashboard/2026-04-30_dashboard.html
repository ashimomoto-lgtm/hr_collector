1on1・マネジメント領域

1on1やコーチングといった対話の質向上が引き続き重要なテーマであり、パーソル総合研究所は「なぜ1on1は機能しないのか？」を、HRzineは「なぜ対話はすれ違うのか？」を紐解く書籍を紹介しています。AIの活用がマネジメント業務に与える影響についても「日本の人事部」が複数調査を実施し、「AI活用が変える職場とマネジメント業務調査」の分析結果を発表しました。リーダーシップの資質や多様性を活かすための行動変革が求められ（プレジデントオンライン「リーダーの資質」、パーソル総合研究所「多様性を活かすためのリーダー行動アップデート」）、フォロワーシップの育成も注目されています。また、「管理職のメンバーに対するイメージ」や「ホワイトハラスメント」など、管理職を取り巻く課題も「日本の人事部」の調査で浮き彫りになっています。小出 翔氏の書籍『誰もが成長し活躍する会社のしくみ』に代表される「スキルベース組織」といった新しい人材マネジメントの実践法も提示され、組織全体の成長を促すマネジメントへの関心が高いです。

人事制度・採用・育成

採用・転職市場ではAI活用への関心が高く、「日本の人事部」が「AI時代の転職・採用に関するアンケート調査」を実施する一方、ロレアルのように採用面接で「AI禁制」とし対面回帰でスキルを見極める動きも見られます。育成面では、新卒研修で生成AIを必修化する動き（日本の人事部）や、NTTドコモが実践するキャリア開発施策など、個人のスキルアップとキャリア自律を支援する取り組みが活発化しています。人事制度の刷新も進んでおり、カルビーが約17年ぶりに人事制度を抜本的に刷新したほか、王子HDは退職一時金を全廃し資産形成制度を導入するなど、人材獲得競争を意識した変化が見られます。京王電鉄が「カオナビ」を導入した事例に代表されるタレントマネジメントシステムの導入も進んでおり、日本企業におけるスキル管理の課題も指摘されています。国際航業の育児・介護休業支援制度「サンクス・シェア」や、ソフトバンクの障害者手帳を持たない人への活躍機会提供など、多様な働き方を支援する包摂的な雇用施策が注目されています。

HRテック・AI活用

HR領域におけるAI活用は急速に進展しており、「日本の人事部」が「AI時代の転職・採用」や「マネジメント業務」に関する調査を複数実施しています。採用プロセスでは、talentbookの「タレントブック・キャンパス」がAIを活用したOB/OG訪問を実現し、JetBの「Our AI面接」やマイナビの「WHAT」がアバター型AI面接やインタラクティブ機能で面接品質向上を図っています。人事業務の効率化では、MIXIがSmartHR導入により就労証明書発行業務の約8割を自動化し、米オラクルも人事業務向け「Fusion Agentic Applications」を発表するなど、DX推進が加速しています。生成AIの職場での普及には課題も指摘されており（パーソル総合研究所「生成AIはなぜ職場に広がらないのか」）、リクルートMSの調査ではAI活用が個人の業務効率化から組織全体の生産性向上へと広がる段階にあることが示唆されています。AI-Readyな企業の条件としてメタデータとガバナンスの重要性が指摘され（三菱UFJリサーチ&コンサルティング）、ギブリーは「Claude Cowork 活用支援」サービスを通じてAIエージェント構築・活用人材の育成にも取り組んでいます。
