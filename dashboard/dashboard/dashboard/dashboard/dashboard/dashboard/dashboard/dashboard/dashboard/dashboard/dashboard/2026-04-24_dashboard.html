1on1・マネジメント領域

1on1やキャリア面談の効果的な運用が引き続き注目されており、パーソル総合研究所のコラムでは「なぜ1on1は機能しないのか？」といった課題提起から「1on1という『旅』を始める」と継続の重要性が語られています。リーダーシップにおいては、ダイヤモンド・オンラインの「仕事ができる上司の当たり前」で具体的な発言や行動が問われ、多様性を活かすためのリーダー行動のアップデートがパーソル総合研究所から提言されています。また、ビズリーチの「Onboard AI」のように、AIを活用してマネージャーのオンボーディング負担を軽減する動きも見られ、マネジメントの質向上と効率化が両輪で進んでいます。組織のエンゲージメント向上や健全な対立を育むための風通しの良い組織作りも重要なテーマです。

人事制度・採用・育成

採用市場では、talentbookの「タレントブック・キャンパス」がAIを活用したOB/OG訪問を実現するなど、AIが選考プロセスに深く関与しつつも、HERPの調査では「人が関わることへの意識は依然高い」という結果が出ています。人事制度においては、freeeの「freee人事評価」やjinjerの統合型人事システム「ジンジャー」のように、人事労務と従業員情報の一元管理を進める動きが活発です。カルビーが約17年ぶりに人事制度を刷新したり、レオパレス21がキャリア形成支援の人事制度改定を行うなど、企業は時代に合わせた制度の見直しを積極的に行っています。また、Schoo for Businessが階層別の年間育成計画パッケージを提供開始するなど、多様な人材の育成支援も強化されています。

HRテック・AI活用

HRテック領域では、AIの活用が急速に進展しており、HRbaseの「労務専門AIエージェント」やOn Technologiesの「WorkOn」のように、人事業務を自律的に行うAIエージェントが登場しています。採用活動においては、talentbookが「タレントブック・キャンパス」でAI時代のOB/OG訪問を実現し、マイナビとMQueが「WHAT」でAI面接を支援、ギブリーはAI面接に「インタラクティブ」機能を追加するなど、AIが選考プロセスの効率化と高度化を推進しています。リクルートMSの調査が示すように、AI活用は個人の業務効率化から組織全体の生産性向上へと広がりを見せており、「AI-Ready」なデータ基盤構築の重要性も指摘されています。企業は入社時研修で生成AIを必修化するなど、AI人材育成にも力を入れています。
