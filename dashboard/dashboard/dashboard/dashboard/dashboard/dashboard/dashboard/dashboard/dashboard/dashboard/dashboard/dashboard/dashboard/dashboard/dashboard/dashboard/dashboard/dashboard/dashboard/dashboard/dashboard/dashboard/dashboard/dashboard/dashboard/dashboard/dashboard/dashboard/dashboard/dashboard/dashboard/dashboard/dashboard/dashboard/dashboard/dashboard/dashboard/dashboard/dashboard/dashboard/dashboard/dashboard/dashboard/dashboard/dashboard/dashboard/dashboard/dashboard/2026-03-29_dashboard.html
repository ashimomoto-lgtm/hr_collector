2 HR界隈トレンド分析

1on1・マネジメント領域

1on1・マネジメント領域では、リーダーシップの質向上が喫緊の課題となっています。「ダイヤモンド・オンライン」では、「心理的安全性」の誤解や「戦略を語れないマネージャー」の問題が指摘されています。上司と部下の「キャリア面談」の重要性が増す一方で、「パーソル総合研究所」のコラムでは管理職が「罰ゲーム化」している現状が浮き彫りになり、その負担軽減が求められています。「日本の人事部」の調査からは、マネジメントに関する意識や組織の「違和感」への対応が、組織の成長に不可欠であることが示唆されています。「パーソル総合研究所」は、人的資本経営において個人の能力向上だけでなく「対話」を通じた組織成果への接続を提言し、風通しの良い組織作りには「健全な対立」の生成が重要だと述べています。

人事制度・採用・育成

人事制度・採用・育成領域では、早期離職対策が重要なテーマです。「日本の人事部」の調査では、新人・若手の早期離職が入社前の情報不足に起因すると指摘されており、入社前の情報提供の質向上が求められています。「パーソル総合研究所」は日本の「離職理由」の変化や、多様性推進における「ダイバーシティ疲れ」の構造的要因を分析し、新しいリテンション・マネジメントの再考を促しています。採用においては、「ニトリ」の「育てる新卒採用」のような取り組みが注目され、学生のやりがいや成長への意識、中途転職者の不満要因の把握が重要です。また、企業型DCの法改正対応（マーサージャパン）や、育児・介護との両立支援、女性登用率向上、ハラスメント判断基準の明確化など、幅広い人事施策が求められています。「HRzine」では、AI時代に企業が育てるべきは「正解を知っている人」ではなく「差を見て判断できる人」であるとの提言もあり、育成手法の見直しも進んでいます。

HRテック・AI活用

HRテック・AI活用の動向は急速に進展しており、「HRzine」の調査では管理職の半数以上が生成AIを活用している実態が明らかになっています。「Unipos」が管理職の「ほめ」をAIでサポートする機能を追加したほか、「TalentX」はAIネイティブATS「MyTalent Hire」を提供し、採用プロセスの自動最適化を図っています。「Workday」は中核システムにAIを組み込んだ「Sana from Workday」を発表し、ビズリーチは学習管理システム「HRMOSラーニング」で人材配置への学習履歴活用を推進しています。「日経ビジネス」はAIが事務仕事の55％を代替し、管理部門が提言型に転換すべきと指摘しており、人事評価（カラフルボックスの「Scale人事評価」）や、コーポレート業務でのAI活用（SCSK）も進んでいます。生成AIが「インフラ化」する中で、業務変革（日本の人事部）や、候補者探索からスカウト生成までAIが実行する機能（YOUTRUST）など、HRのあらゆる場面でのAI活用が加速しています。
