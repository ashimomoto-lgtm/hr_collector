1on1・マネジメント領域

1on1やマネジメントの質向上が喫緊の課題となっています。「パーソル総合研究所」のコラムでは、1on1が機能しない理由や部下の考える力を育てる対話の再設計が繰り返し議論されています。また、「日本の人事部」の調査では管理職のメンバーに対するイメージやマネジメントの状態が分析され、サーベイデータを「兆し」として捉える重要性が指摘されています。リーダーシップにおいては、部下との関係構築、多様性を活かすリーダー行動のアップデートが求められており、「ダイヤモンド・オンライン」の記事では「優しいけど、嫌われるリーダー」の特徴が挙げられています。さらに、「ビズリーチ」が提供する「Onboard AI」のように、AIを活用してマネージャーのオンボーディング負担を軽減する動きも見られます。

人事制度・採用・育成

人事制度・採用・育成領域では、人材獲得の困難が主要なリスクとして認識されており、多様な採用・育成手法が展開されています。「ニトリ」が通年採用を導入し、「日本の人事部」がカジュアル面談の実態調査を行うなど、採用活動の柔軟性が高まっています。「カルビー」が約17年ぶりに人事制度を刷新し、「NECビジネスインテリジェンス」がジョブ型人材マネジメントの次の一手を模索するなど、既存の人事制度を見直す動きが活発です。「Schoo for Business」による階層別育成計画パッケージの提供や、経産省の「デジタルスキル標準ver.2.0」公表など、従業員のスキルアップとキャリア自律支援が重視されています。また、物価上昇局面でのベア優先や、シニア社員・公務員の副業、地方における人材確保など、多様な働き方や報酬制度への関心が高まっています。

HRテック・AI活用

HRテックとAIの活用は、人事領域の広範囲で進化を遂げています。「マイナビ」と「MQue」がAI面接支援サービス「WHAT」を提供開始し、「ギブリー」はAI面接にインタラクティブ機能を追加するなど、採用活動におけるAI活用が高度化しています。「freee人事評価」や「jinjer」の統合型人事システム「ジンジャー」のように、人事労務と従業員情報を一元管理するHRテックソリューションの導入が進んでいます。「On Technologies」の「WorkOn」や「HRbase」の「労務専門AIエージェント」のように、人事業務を自律的に行うAIエージェントの進化も見られます。「リクルートMS」の調査や「三菱UFJリサーチ&コンサルティング」の記事が示すように、AI活用は個人の業務効率化に留まらず、組織全体の生産性向上、そして「AI-Ready」なデータ基盤とガバナンス構築が重要視されています。
