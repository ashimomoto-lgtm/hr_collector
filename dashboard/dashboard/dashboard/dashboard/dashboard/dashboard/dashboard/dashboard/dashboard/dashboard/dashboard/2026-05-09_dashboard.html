2 HR界隈トレンド分析

1on1・マネジメント領域

1on1や対話の重要性が認識されつつも、「なぜ機能しないのか？」という課題意識が強く、「部下の考える力を育てる対話の再設計」や「キャリア面談の効果を高める」といった実践的なアプローチが求められています（パーソル総合研究所）。「AI時代のマネジメント業務と学習環境に関する実態調査」や「AI活用が変える職場とマネジメント業務調査」が示すように、AIがマネジメント業務に与える影響への関心が高まっています。リーダーシップにおいては、多様性を活かすための行動アップデートやフォロワーシップの育成が注目され、「建設的なコンフリクト」を育む組織作りが重視されています（パーソル総合研究所）。また、個人のセルフマネジメント能力向上も、成長する人材の条件として挙げられています。

人事制度・採用・育成

人事制度・採用・育成領域では、企業が「人的資本経営」を強く意識し、「社員の声と組織に関する意識」調査（日本の人事部）や「従業員意識調査」を通じて従業員エンゲージメントの向上を図っています。採用においては、「AI時代の転職・採用に関するアンケート調査」や「企業の新卒採用実態調査2026」が示すように、AIの活用が進む一方で、ミスマッチを防ぐための「リファレンスチェック」の重要性が指摘されています（HRzine）。育成・制度面では、カルビーが約17年ぶりに人事制度を刷新し、NTTドコモがキャリア開発施策を実践するように、個人の成長と企業の成長を両立させる「スキルベース組織」や「ジョブ型人材マネジメント」への移行が注目されています。国際航業の「サンクス・シェア」制度のように、多様な働き方を支える新しい制度導入も進んでおり、ダイバーシティ経営推進への具体的なアクションが求められています。

HRテック・AI活用

HRテック・AI活用の領域では、AIが採用からマネジメント、スキル評価に至るまで、HR業務の広範囲に浸透しつつあります。「AI時代の転職・採用に関するアンケート調査」や「AI時代のマネジメント業務と学習環境に関する実態調査」が示すように、AIがもたらす変化への関心は高く、川崎重工業がAIキャリアエージェント「BaseMe」を導入し、SCSKが生成AIを活用したスキル評価システムを開発するなど、具体的な導入事例が増えています。採用プロセスでは、「PeopleX AI面接」や「Our AI面接」といったAI面接サービスが企業と候補者の適合度を自動判定し、効率化と質の向上を目指しています。また、京王電鉄がタレントマネジメントシステム「カオナビ」を導入したように、人材情報の可視化・活用も活発です。一方で、「生成AIはなぜ職場に広がらないのか」といった調査（パーソル総合研究所）もあり、AI活用の実態と課題も浮き彫りになっています。
