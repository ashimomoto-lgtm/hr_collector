2 HR界隈トレンド分析

1on1・マネジメント領域

1on1やキャリア面談の重要性が改めて注目されており、パーソル総合研究所のコラム「1on1という『旅』を始める」では、その継続の難しさに触れつつも、上司と部下の対話を通じて部下のキャリア自律を促す動きが見られます。多様化する組織において、リーダーシップのあり方も変化しており、パーソル総合研究所の「多様性を活かすためのリーダー行動アップデート」では、多様性を活かすためのリーダー行動が考察されています。管理職の負担増大、いわゆる「罰ゲーム化する管理職」といった課題も指摘される中で、風通しの良い組織作りやOJTの仕組み化が求められています。また、人的資本経営を一歩先へと進める「対話」の重要性も強調されています。

人事制度・採用・育成

人的資本経営の推進に伴い、個人のキャリア自律支援や対話を通じた組織成果への接続が重視されています。カルビーが約17年ぶりに人事制度を刷新したように、多様な貢献と成長を促すための制度改革が進む一方、英ユニリーバの採用凍結など、経済状況が採用に影響を与えるケースも散見されます。賃上げ局面における報酬設計の盲点や、インフレ・人材不足時代における「給与の未来展望」など、リテンション・マネジメントにおける報酬制度の役割が再評価されています。また、20代若手社員のスピード退職や就活セクハラ実態調査など、採用・定着における新たな課題も顕在化しており、ワークライフバランスや女性の幸福度に関する調査も活発です。

HRテック・AI活用

HRテックの導入は広がりを見せており、ANAグループが「SmartHR」を導入して業務効率化とタレントマネジメントを推進するなど、基幹システムとしての活用が進んでいます。特にAIの活用が顕著で、三井住友信託銀行が「AI面接官」を導入して公平・多様な採用プロセスを実現したり、「HRBrain Brain」がAI新機能で人事データ分析を高度化したりしています。日本の人事部では「入社時研修で"生成AI"を必修化」する企業も登場し、管理職や若手社員のAI活用実態調査も進んでいます。一方で、パーソル総合研究所のコラム「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」が示すように、AI導入だけでは解決しない課題や、就活会議が生成AI悪用対策を導入するなど、AI活用の光と影の両面が浮き彫りになっています。
