2 HR界隈トレンド分析

1on1・マネジメント領域

1on1やキャリア面談の重要性が再認識されており、パーソル総合研究所のコラム「1on1という「旅」を始める」ではその継続的な取り組みが強調されています。多様な人材を活かすためのリーダー行動のアップデートが求められ（パーソル総合研究所「多様性を活かすためのリーダー行動アップデート」）、旧来の若者観に囚われないマネジメントの必要性も指摘されています（ダイヤモンド・オンライン「あなたの違和感は正しい。「最近の若いやつ」で片づける人ほどハマる、危ない思い込み」）。また、中間管理職の「罰ゲーム化」という課題が指摘される中で（パーソル総合研究所「罰ゲーム化する管理職」）、組織改革やサクセッションプランの策定・運用を通じた「強いミドル」の復活が模索されています。人的資本経営においては、個人の能力向上だけでなく「対話」を通じて組織全体の成果につなげる視点が重視されています（パーソル総合研究所「個人の人的資本を高めるだけでは、組織の成果につながらない？」）。

人事制度・採用・育成

人材の離職防止とリテンション・マネジメントが喫緊の課題となっており、日本の人事部による「退職理由の本音と建前に関する実態調査」や、パーソル総合研究所の「給与の未来展望」に関するコラムがその背景を深く掘り下げています。人事制度の抜本的な刷新が進んでおり、カルビーが約17年ぶりに新制度を導入し、多様な貢献と成長を促す取り組みが見られます。採用領域では、ベンチャー企業が「集中突破」戦略で優秀層を獲得する事例（HRzine「ベンチャー企業の採用勝ちパターン」）や、転職市場における若手社員のスピード退職動向（日本の人事部「20代若手社員のスピード退職と転職動向に関する調査」）が注目されています。キャリア自律支援やジョブ型人材マネジメントへの移行（パーソル総合研究所「ジョブ型人材マネジメントの「次の一手」を考える」）、さらにはシニア社員の副業位置付けなど、多様な働き方とキャリア形成を支える制度設計が求められています。インフレ時代における賃上げのあり方や、女性登用率の向上も重要なテーマです。

HRテック・AI活用

HRテックとAIの活用は人事領域で急速に拡大しており、生成AIは入社時研修での必修化（日本の人事部「入社時研修で"生成AI"を必修化」）や、課長層の活用実態調査（日本の人事部「課長のAI活用実態調査」）が示すように、組織全体に浸透しつつあります。採用プロセスではAI面接官の導入が進み、三井住友信託銀行が「VARIETAS」を導入して公平性を高め、レバレジーズも「NALYSYS AI面接」で面接の属人化解消を図っています。タレントマネジメントシステムも進化し、「SmartHR」がANAグループに導入され、HRBrainはAI新機能「HRBrain Brain」で高度な人事データ分析を実現しています。しかし、生成AIを仕事で使っても忙しさが変わらないという課題意識も存在し（パーソル総合研究所「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」）、AIが代替できない「アンテナ」や「信頼貯金」の重要性も指摘されています（HRzine「AIは人事労務をどう"進化・深化"させるのか？」）。就職活動においてもAIが活用され、学生がAIの提案する未知の業界に関心を持つなど新たな動向が見られますが、生成AIを悪用した虚偽投稿への対策も必要とされています（就活会議「生成AIを悪用した虚偽投稿への対策を導入」）。
