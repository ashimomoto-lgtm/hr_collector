2 HR界隈トレンド分析

1on1・マネジメント領域

1on1・マネジメント領域では、対話の質向上とリーダーシップの変革が主要な動向です。プレジデントオンラインの「部下との1on1は『お願いします』で始めてはいけない」やパーソル総合研究所の「1on1という『旅』を始める」は、効果的な1on1の重要性を強調しています。また、日経ビジネスの入山章栄教授の講演やプレジデントオンラインの「心理的安全性」に関する記事は、次世代リーダーの育成や多様性を活かすための上司の行動アップデートが求められていることを示唆しています。パワハラ防止や「ジョブ型人材マネジメント」の進化も重要なテーマであり、実践的なマネジメント手法への関心が高まっています。

人事制度・採用・育成

人事制度・採用・育成領域では、採用市場の変化が顕著です。日経ビジネスの「AI時代、『採用減る』が4割　新卒は必要か」という特集記事は、「脱・大量採用」や「新卒厳選・中途拡大」の潮流を伝えています。HRzineのuloqo調べでは「カジュアル面談で志望度が下がった経験」が3人に1人という結果が出ており、採用ブランディングの重要性が増しています。育成面では、日本の人事部が報じる入社時研修での「生成AI」必修化や、Skillnoteの「個人カルテ」機能追加など、個人のスキル開発とキャリア自律支援が強化されています。さらに、ストレスチェック義務化の拡大や高年齢者の労働災害防止指針の適用など、多様な働き方への対応と安全衛生の確保も重要な課題となっています。

HRテック・AI活用

HRテック・AI活用の動向として、AIの導入が急速に進んでいます。AGCが「AI面接官」を導入した事例（HRzine）は、採用プロセスの公平性と効率化への期待を示しています。マネーフォワードの「AI Cowork」やキャスターの「AI社員」のように、AIがバックオフィス業務を自律的に遂行するサービスも登場し、業務効率化への期待が高まっています。一方で、パーソル総合研究所のコラム「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」では、AI導入後の実態と課題も提起されており、効果的な活用に向けた模索が続いています。ビズリーチの「HRMOS労務」やfreeeの統合型システムなど、HR業務を一気通貫でサポートするクラウドサービスの進化も注目されています。
