1on1・マネジメント領域

1on1は依然として重要なマネジメント手法ですが、その機能不全が指摘され、「なぜ1on1は機能しないのか？」（パーソル総合研究所）のような再考が促されています。リーダーシップにおいては、不確実な時代に対応できる「責任を引き受けるリーダー」（日経ビジネス）の育成や、多様性を活かすためのリーダー行動のアップデートが求められています。管理職のマネジメント実態に関する調査（日本の人事部）からは、健全なコンフリクトを育む「風通しの良い組織」作りが重要視されており、部下の考える力を育てる対話の再設計が課題です。また、「サクセッションプランの策定と運用」（三菱UFJリサーチ&コンサルティング）に見られるように、リスクマネジメントの観点から経営リーダーの育成や組織のケイパビリティ強化への関心も高まっています。

人事制度・採用・育成

採用活動では「カジュアル面談に関する実態調査」（日本の人事部）が実施される一方、「オワハラ」のような悪質な内定辞退阻止の動きも報じられており、学生の就活環境は多様化しています。人事制度においては、カルビーが約17年ぶりに制度を刷新し、レオパレス21もキャリア形成と働きやすさを支援する改定を行うなど、企業が従業員のキャリア自律とエンゲージメント向上に注力しています。育成面では、ポストAI時代を見据えたキャリア支援の再設計が議論され、「Schoo for Business」のように階層別の年間育成計画パッケージが提供されるなど、体系的な学びの機会が重視されています。「人材流失、人材獲得の困難による人材不足」が優先リスクの1位（デロイト調べ）とされ、採用競争の激化がうかがえるほか、「静かな退職」や「ホワイトハラスメント」に関する調査（日本の人事部）も行われ、従業員の心理的安全性への配慮が人事課題として浮上しています。

HRテック・AI活用

HRテック領域では、AIを活用したサービスが急速に拡大しており、ビズリーチの「Onboard AI」によるオンボーディング支援や、On Technologiesの「WorkOn」による人事業務の自律化など、多岐にわたるソリューションが登場しています。タレントマネジメントシステムでは、兵庫県が「タレントパレット」を導入し、湖池屋が統合型人事システム「ジンジャー」で人事労務を一元化するなど、データに基づいた人事戦略へのシフトが進んでいます。AI活用は採用・育成にも広がり、26卒・27卒の「就活・AI活用実態調査」（日本の人事部）や、パーソルHDが管理職の目標設定支援AIエージェントを開発する事例が見られます。企業におけるAI活用の実態調査（日本の人事部）や、入社時研修での生成AI必修化（日本の人事部）など、AIリテラシーの向上が急務と認識されています。一方で、サーベイや行動データの取り扱い方（HRzine）や、AIを活かすためのメタデータとガバナンス（三菱UFJリサーチ&コンサルティング）の重要性も指摘されており、技術導入だけでなく、その適切な運用が問われています。
