人事制度・採用・育成

人事制度においては、カルビーが約17年ぶりに人事制度を抜本的に刷新するなど、企業が変革期を迎えています。採用活動では「企業の新卒採用実態調査2026」や「中途採用における求職者の情報収集行動」が実施され、AI時代の採用活動における求職者と企業の双方の意識が調査されています。育成面では、入社時研修で生成AIを必修化する動きが見られ、NTTドコモのような企業がキャリア開発施策を実践し、社員のキャリア形成に熱心な企業が注目されています。また、国際航業が育児・介護休業を支える「サンクス・シェア」制度を導入するなど、多様な働き方を支援する制度が広がっています。

HRテック・AI活用
