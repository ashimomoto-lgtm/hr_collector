1on1・マネジメント領域

1on1やマネジメント領域では、その形骸化が課題視され、上司の「アドバイスする」姿勢の再考や、部下の考える力を育む対話の再設計が求められています（ダイヤモンド・オンライン「残念ですが部下に信頼されません…1on1で『アドバイスする上司』の致命的な勘違い」、パーソル総合研究所「なぜ1on1は機能しないのか？」）。多様性を活かすためのリーダー行動のアップデートや、フォロワーシップ、健全なコンフリクトを育むマネジメントの重要性が強調されており（パーソル総合研究所「多様性を活かすためのリーダー行動アップデート」、「建設的なコンフリクトをどう育てるか」）、対話の質を高める書籍も注目されています（HRzine「1on1、コーチング——『なぜ対話はすれ違うのか？』を紐解く"対話の時代"必読の書」）。また、AI時代のマネジメント業務に関する実態調査も進み（日本の人事部「AI時代のマネジメント業務と学習環境に関する実態調査」）、テクノロジーと共存する新しいリーダーシップのあり方が模索されています。

人事制度・採用・育成

人事制度・採用・育成領域では、キャリア自律を促す取り組みが活発化しており、NTTドコモが「自分らしさ」と「会社の成長」を両立させるキャリア開発施策を実践しています（パーソル総合研究所「NTTドコモが実践するキャリア開発施策」）。カルビーが約17年ぶりに人事制度を抜本的に刷新した事例や、「スキルベース組織」への移行が提唱されるなど、人事制度の刷新も進んでいます（日本の人事部「カルビー、約17年ぶりに人事制度を抜本的に刷新」、書籍『誰もが成長し活躍する会社のしくみ 「スキルベース組織」という新しい人材マネジメントの実践法』）。採用においては、新卒・中途採用の実態調査が頻繁に行われ（日本の人事部「企業の新卒採用実態調査2026」、「中途採用における求職者の情報収集行動」）、Z世代の仕事観やクォーターライフクライシスといった若年層の課題への関心が高まっています（日本の人事部「Z世代の仕事観と自分らしさに関する調査2026」、「正社員のクォーターライフクライシス調査2026年」）。国際航業が育児・介護休業を支える新制度「サンクス・シェア」を導入するなど、多様な働き方を支援する制度設計も進展しています。

HRテック・AI活用

HR領域におけるAI活用が急速に進展しており、採用ではアバター型AI面接サービス「Our AI面接」（JetB）や、候補者の回答を深掘る「インタラクティブ」機能を追加したギブリーのAI面接サービスが登場しています。On Technologiesが人事業務を自律的に行うAIエージェント「WorkOn」を提供開始するなど、業務効率化・自動化の動きが顕著です。企業は「AI時代の転職・採用」や「AI時代のマネジメント業務」に関する実態調査を通じて、AIが職場や業務に与える影響を分析しています（日本の人事部）。一方で、生成AIが職場に広がらない課題や、利用率だけでは測れない活用の実態も指摘されており（パーソル総合研究所「生成AIはなぜ職場に広がらないのか」、「利用率では測れない、職場での生成AI活用の実態」）、AIを活かすためのメタデータとガバナンスの重要性も議論されています（三菱UFJリサーチ&コンサルティング「AI-Readyな企業の条件とは」）。
