2 HR界隈トレンド分析

1on1・マネジメント領域

上司と部下の「対話」を通じたキャリア面談や人的資本経営の推進が重視されており、パーソル総合研究所のコラム「個人の人的資本を高めるだけでは、組織の成果につながらない？」がその重要性を指摘しています。リーダーシップにおいては、東洋経済オンラインが「正論より"物語"がリーダーに必要な決定的な理由」を説き、組織の活性化には「戦略を持たない管理職」の課題解決が求められています。また、パーソル総合研究所のコラム「罰ゲーム化する管理職」が示すように、ミドルマネジメント層の負担増が課題となっており、グーグルのマネジメントプログラムのような先進事例から学び、OJT実践マニュアルを活用するなど、リーダーシップ開発と組織改革への取り組みが活発です。三菱UFJリサーチ&コンサルティングによる「サクセッションプランの策定と運用」も、リスクマネジメントの観点から注目されています。

人事制度・採用・育成

離職率の高まりを受け、パーソル総合研究所のコラム「日本の『離職理由』はどう変化したのか」が示すように、リテンションマネジメントが重要視されています。賃上げ局面ではパーソル総合研究所のコラム「給与の未来展望」や「報酬設計の盲点」が議論され、インフレ・人材不足時代における人材つなぎとめ策が課題です。採用においては「就活セクハラに関する実態調査」が示すように倫理的な問題への対応が求められる一方、「27卒学生の就活動向」調査も進んでいます。また、エムステージグループの「治療と仕事の両立支援制度」やダスキンの「パートナーシップ制度」など、多様な働き方を支援する人事制度の導入が進み、NECビジネスインテリジェンスの事例に学ぶ「ジョブ型人材マネジメント」への関心も高まっています。

HRテック・AI活用

HRテック分野では、生成AIの活用が急速に進んでいます。日本の人事部が報じるように、企業は入社時研修で「生成AI」を必修化する動きを見せており、Uniposは管理職の「ほめ」をAIがサポートする機能を追加しました。パーソル総合研究所のコラム「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」が問いかけるように、AIの導入効果を最大化する課題も浮上しています。また、エフアンドエムが人事労務マニュアルに基づくAI回答機能をリリースするなど、業務効率化へのAI活用が進む一方、就活会議では「生成AIを悪用した虚偽投稿への対策」も導入されており、倫理的な利用が求められています。ビズリーチの「HRMOSラーニング」やHRBrainの「採用ソリューション」のように、学習管理や採用プロセスにおけるHRテックの導入も活発です。
