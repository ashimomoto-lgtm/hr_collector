2 HR界隈トレンド分析

1on1・マネジメント領域

1on1の形骸化が課題とされており、「なぜ1on1は機能しないのか？」（パーソル総合研究所）といった問いかけや、部下の考える力を育てる対話の再設計、キャリア面談の効果向上など、その質を高めるための模索が続いている。管理職の能力や自己評価と実態の乖離が指摘されており、『上司はリスクばかりを指摘する 会社を潰す「大課長」問題』などの書籍が刊行され、マネジメント層への警鐘が鳴らされている。多様な人材を活かすためのリーダー行動のアップデートや、風通しの良い組織を作るための「健全な対立」の生成が重要視されている（パーソル総合研究所）。ジョブ型人材マネジメントの推進や、離職理由の変化に対応したリテンション・マネジメントなど、戦略的なマネジメントの必要性が高まっている。

人事制度・採用・育成

企業は人事制度の抜本的な刷新を進めており、カルビーが約17年ぶりに人事制度を刷新したように、時代に合わせた制度設計が加速している。子育てとキャリアの両立支援（EYSCによる電通ランウェイへの支援）や、従業員の学びを促進するカフェテリアプラン（MIXIの「発明補助」ポイント）など、多様な働き方を支える福利厚生や育成制度が導入されている。採用においてはカジュアル面談が普及し、28卒の夏インターンシップ参加予定社数に関する調査も行われるなど、学生との接点づくりが早期化・多様化している。転職市場は活発で、20代若手社員のスピード退職や中途採用が多い企業ランキングが注目される一方、公務員やシニア社員の副業に関する議論も進んでいる。ベースアップに関する意識調査や、ストレスチェック義務化の拡大（ニッセイ基礎研究所）など、賃金や従業員の健康・安全に関する制度面での配慮も強化されている。

HRテック・AI活用

AI活用はHR領域で急速に拡大しており、イオングループやAGCが「PeopleX AI面接」「AI面接官」を導入するなど、採用プロセスでのAI活用が本格化している。しかし、AI面接に対する学生の抵抗感や個人情報のリスク、人事担当者の「AIで新卒採用減」という懸念も浮上しており、倫理的・実務的な課題も顕在化している（日経ビジネス）。企業は「AI-ready」な組織への変革を急ぎ、入社時研修での生成AI必修化や、AI戦略を絵に描いた餅で終わらせないための要員計画策定が求められている（三菱UFJリサーチ&コンサルティング）。HRテックサービスも多様化し、「HRMOS労務」（ビズリーチ）のような統合型プラットフォームや、「freee業務委託管理」「弥生勤怠 Next」といったクラウドサービスが提供され、業務効率化とデータ活用を推進している。若手社員のキャリア観や課長のAI活用実態調査など、AIが働き方やキャリアに与える影響に関する関心も高く、その効果的な活用方法が模索されている。
