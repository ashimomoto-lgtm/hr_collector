1on1・マネジメント領域

1on1やキャリア面談の質向上が喫緊の課題であり、パーソル総合研究所のコラム「なぜ1on1は機能しないのか？」や「上司と部下のキャリア面談の効果をより高めるには？」がその再設計の重要性を説いています。多様な人材を活かすためのリーダー行動変革が求められ、パーソル総合研究所の「多様性を活かすためのリーダー行動アップデート」や「建設的なコンフリクトをどう育てるか」が具体的な指針を示しています。また、フォロワーシップの育成や風通しの良い組織作りも重視されており、書籍「日本一の不器用が手に入れた『最強の組織』」や「『管理職のメンバーに対するイメージ』とマネジメントの状態に関する調査」が組織活性化への関心の高まりを裏付けています。

人事制度・採用・育成

採用市場ではカジュアル面談や中途採用における求職者の情報収集行動に注目が集まり、「カジュアル面談に関する実態調査」や「中途採用における求職者の情報収集行動」が実施されています。企業は多様な人材獲得に力を入れており、SCREENが「博士採用」を強化し「カジュアル面接」で接点を増やす事例や、セブン銀が「障害者採用にもインターン」を導入する動きが見られます。人事制度の抜本的見直しも進み、カルビーが約17年ぶりに人事制度を刷新したほか、ジョブ型人材マネジメントへの関心も高まっています。また、入社時研修での"生成AI"必修化や「デジタルスキル標準ver.2.0」の公表など、デジタル人材の育成が喫緊の課題です。「正社員の静かな退職に関する調査」や「ホワイトハラスメントに関する調査」からは、従業員のエンゲージメントや職場環境改善への継続的な関心がうかがえます。

HRテック・AI活用

HRテック領域では、AIの活用が採用・育成・マネジメントの各フェーズで加速しています。talentbookの「タレントブック・キャンパス」によるAI時代のOB/OG訪問や、ビズリーチの「Onboard AI」によるオンボーディング支援など、AIを活用したサービスが多様化しています。採用プロセスでは、マイナビが「WHAT」でAI面接支援を、ギブリーが「インタラクティブ」機能を追加したAI面接を提供し、候補者の見極めを高度化しています。タレントマネジメントシステム導入も活発で、京王電鉄が「カオナビ」を、兵庫県が「タレントパレット」を導入し、データに基づいた人事戦略を推進しています。しかし、パーソル総合研究所の「生成AIはなぜ職場に広がらないのか」や「利用率では測れない、職場での生成AI活用の実態」が示すように、職場での生成AI普及にはまだ課題があり、AI活用の成功には「AI-Readyなデータ基盤」の構築が不可欠であると指摘されています。
