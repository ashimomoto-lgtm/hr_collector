1on1・マネジメント領域

1on1の効果的な運用とマネージャーの役割に焦点が当たっており、「パーソル総合研究所」のコラムでは、「なぜ1on1は機能しないのか？」や「部下の考える力を育てる対話の再設計」が議論されています。単なる目標設定ではなく、部下の自律的な成長を促す「問いかけ」（プレジデントオンライン）が重要視される傾向にあります。「ダイヤモンド・オンライン」が指摘するように、チームを強くするだけでなく、多様性を活かし「健全な対立」を育むリーダー行動（パーソル総合研究所）が、不確実な時代における経営リーダー育成や組織の「勝ち方」として求められています。管理職の負担軽減や、部下へのイメージとマネジメントの実態に関する調査も行われ、より実践的なマネジメント手法への関心が高まっています。

人事制度・採用・育成

採用・育成においては、カジュアル面談の実態調査や「タレントプール」の活用（日立）、「アルムナイ採用制度」（J:COM）の新設など、多様な人材獲得チャネルが模索されています。人事制度では、カルビーが約17年ぶりに抜本刷新し、子育てとキャリアの両立支援（EYSC）や、従業員が自律的にお金の問題を解決できる新サービス「お金のEAP」の提供など、従業員のエンゲージメントとウェルビーイング向上に繋がる動きが活発です。「リクルートMS」の調査では約3分の2の企業が採用手法の見直しを実感しており、事業戦略と連動した「タレントマネジメントの推進力」（大阪ガス）や、「ジョブ型人材マネジメント」（NECビジネスインテリジェンス）への関心が高まっています。ベースアップに関する意識調査やストレスチェック義務化の対象拡大（ニッセイ基礎研究所）など、労働環境の改善と法令遵守も重要なテーマです。

HRテック・AI活用

HRテックとAIの活用は急速に進展しており、「イオングループ」が「PeopleX AI面接」を導入するなど、採用プロセスでのAI活用が本格化しています。「パーソルHD」が管理職の目標設定支援AIエージェントを開発し、「ClipLine」が「すかいらーくレストランツ」に現場向けAIエージェント「ABILI Pal」を導入するなど、社内業務効率化やマネジメント支援にもAIが活用されています。一方で、「日経ビジネス」が指摘するようにAI活用による企業間格差が広がる中、「三菱UFJリサーチ&コンサルティング」は「AI-Readyな企業」の条件としてメタデータとガバナンスの重要性を強調しています。就活生や若手社員、50・60代の生成AI活用実態調査も進み、「ニッセイ基礎研究所」は大学生の生成AIに対する不安を指摘するなど、導入だけでなく、その影響と適切な利活用が今後の課題となっています。
