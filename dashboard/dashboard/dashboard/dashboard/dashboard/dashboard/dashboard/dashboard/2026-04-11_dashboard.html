1on1・マネジメント領域

1on1の機能不全が指摘される中で、パーソル総合研究所のコラム「なぜ1on1は機能しないのか？」が示すように、部下の考える力を育む対話の再設計が求められています。上司の行動変容が組織変革の鍵とされ、ダイヤモンド・オンラインの「【部下が変わらないのは上司のせい】」ではリーダーシップの重要性が強調されています。また、キャリア面談の効果向上や多様性を活かすリーダー行動、風通しの良い組織作りが焦点となり、書籍『世界の研究者はマネジメントをどう分析しているのか』なども注目されています。離職理由の変化に対応するリテンション・マネジメントや、ジョブ型人材マネジメントの「次の一手」に関する議論も活発です。

人事制度・採用・育成

新卒採用は変革期を迎え、日経ビジネスの「AI時代、『採用減る』が4割 新卒は必要か」が示すように、富士通やJALが新卒一括採用を見直し、厳選採用や中途拡大へとシフトしています。カジュアル面談での志望度低下がuloqoの調査で明らかになるなど、採用プロセスにおける応募者体験の質が重視されています。育成面では、パーソル総合研究所のコラム「上司と部下のキャリア面談の効果をより高めるには？」が示すようにキャリア自律支援が重要視され、高年齢者の労働災害防止指針など安全衛生・ウェルビーイング制度整備も進んでいます。地方での人材誘致や副業・兼業の推進、物価上昇に対応したベア優先など、多様な働き方や報酬制度への対応が人事制度の主要課題となっています。

HRテック・AI活用

HR領域でのAI活用が急速に進展し、「26卒・27卒【就活・AI活用実態調査】」が示すように、就職活動や課長の業務にAIが浸透しています。企業ではAGCが「AI面接官」を導入し、東京海上日動がTalentXの「MyTalent Platform」を導入するなど、採用プロセスでのテクノロジー活用が加速しています。freeeやマネーフォワードはAIによるバックオフィス業務サポートを強化し、ビズリーチは「HRMOS労務」をリリースするなど、HRテックサービスが業務効率化を推進しています。一方で、パーソル総合研究所のコラム「生成AIを仕事で使っても、なぜ私たちは忙しいままなのか」が指摘するように、AI導入後の効果的な活用や働き方への影響には依然として課題が残されています。
