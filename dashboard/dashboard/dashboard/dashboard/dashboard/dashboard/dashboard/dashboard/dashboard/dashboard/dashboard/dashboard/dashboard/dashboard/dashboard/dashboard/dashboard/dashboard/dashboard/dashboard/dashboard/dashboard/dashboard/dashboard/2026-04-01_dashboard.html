1on1・マネジメント領域

1on1は依然として重要なマネジメント手法であり、パーソル総合研究所のコラム「1on1という『旅』を始める」ではその継続的な取り組みの重要性が説かれています。リーダーシップのあり方も進化しており、ダイヤモンド・オンラインの「成果を出すリーダーは『台本』と『アドリブ』を使いこなす」が示すように、柔軟性と戦略性が求められています。管理職の役割は「罰ゲーム化する管理職」との指摘（パーソル総合研究所）がある一方で、パーソル総合研究所の「個人の人的資本を高めるだけでは、組織の成果につながらない？　～人的資本経営を一歩先へと進める『対話』のすすめ～」のように、対話を通じた人的資本経営の推進が強調されています。パワハラ認定（HRzine）や組織改革の必要性（日本の人事部「なぜ日本企業は変われないのか？」）など、マネジメントにおけるコンプライアンスと変革への意識も高まっています。風通しの良い組織作りや健全な対立の生成（パーソル総合研究所「風通しの良い組織を作る4つのポイント」）といった、組織文化とコミュニケーションの質を高める取り組みが注目されています。

人事制度・採用・育成

採用市場では、20代若手社員のスピード退職や転職動向（日本の人事部）が注目され、企業はリテンション・マネジメント（パーソル総合研究所「日本の『離職理由』はどう変化したのか」）の強化を迫られています。報酬制度においては、インフレ・人材不足時代に対応した給与の未来展望（パーソル総合研究所「インフレ・人材不足時代に人材をつなぎとめる『給与の未来展望』」）や、賃上げ局面での報酬設計の盲点（パーソル総合研究所）が課題となっています。育成面では、グーグルのマネジメントプログラムの無料受講（日本リスキリングコンソーシアム）や、日揮HDにおけるAI進化をにらんだ人材育成（HRzine）など、リスキリングやDXに対応した教育が活発です。多様な働き方や社員のウェルビーイングを支援する動きも加速しており、エムステージグループの「治療と仕事の両立支援制度」やダスキンの「パートナーシップ制度」、ワーキングケアラー支援（日本の人事部）などがその例です。ジョブ型人材マネジメント（パーソル総合研究所「ジョブ型人材マネジメントの『次の一手』を考える」）やサクセッションプラン（三菱UFJリサーチ&コンサルティング）の策定・運用を通じて、戦略的な人材配置と育成が重視されています。

HRテック・AI活用

HR領域におけるAI活用は急速に進展しており、日本の人事部が実施した「若手社員のキャリア観と生成AI活用に関する調査」や「管理職1000名調査」からも、その浸透度が伺えます。採用活動では、KAGUYAの「採用戦略AIーPersona Voice for HRー」のようなAIツールが登場し、就活におけるAI活用実態調査（No Company）では、AIが提案する未知の業界に学生が関心を持つ傾向も明らかになりました。タレントマネジメントシステムでは、カオナビが評価ワークフローに相対評価の自動調整機能を追加するなど、既存システムのAI連携・機能強化が進んでいます。AIは人事労務業務の効率化にも寄与し、エフアンドエムがリリースした就業規則などに基づくAI回答機能や、レビックグローバルのSmartSkill CampusにおけるAI分析機能などがその例です。一方で、生成AIの悪用対策（就活会議）や、AIが代替できない「アンテナ」「信頼貯金」（HRzine）といった人間的要素の重要性も指摘されており、AIと人間の協調が今後の焦点となります。
