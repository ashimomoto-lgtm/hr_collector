人事制度・採用・育成領域では、企業が採用手法の見直しを迫られており、「リクルートMS」の「企業の新卒採用実態調査2026」では約3分の2の企業が変革の必要性を感じています。カルビーが約17年ぶりに人事制度を抜本的に刷新し、J:COMはアルムナイ採用制度を新設するなど、時代に合わせた制度の見直しと多様な人材獲得戦略が進展しています。若手社員のキャリア観や人的資本開示への関心が高まり、子育てとキャリアの両立支援、ストレスチェック義務化の拡大など、従業員のウェルビーイングや多様な働き方を支える制度設計が重視されています。ジョブ型人材マネジメントの推進やベースアップに関する意識調査も行われ、報酬制度の適正化も引き続き焦点となっています。

HRテック・AI活用は急速に進展しており、イオングループが対話型AI面接サービス「PeopleX AI面接」を導入するなど、採用プロセスでのAI活用が具体化しています。「すかいらーくレストランツ」が現場向けAIエージェント「ABILI Pal」を2600店舗に導入し、JALも「Advanced Teaming Experience Service powered by UMU」を導入しており、AIが業務効率化や学習支援に活用されています。大学生の就活や50・60代の社員におけるAI活用実態調査も行われ、幅広い世代でのAIリテラシー向上と活用が課題となっています。人事部門が「AIネイティブ組織」を目指し、AI戦略と要員計画を連動させる動きや、AI活用による企業間格差の拡大への警鐘も鳴らされており、戦略的なAI導入が求められています。
