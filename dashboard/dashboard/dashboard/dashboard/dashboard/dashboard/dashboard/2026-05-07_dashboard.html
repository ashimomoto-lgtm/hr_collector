2 HR界隈トレンド分析

1on1・マネジメント領域

1on1は依然として重要な対話手法であるものの、「なぜ対話はすれ違うのか？」（HRzine）、「なぜ1on1は機能しないのか？」（パーソル総合研究所）といった課題意識が強く、その再設計が求められています。AI時代におけるマネジメント業務の実態調査が行われ（日本の人事部「AI時代のマネジメント業務と学習環境に関する実態調査」）、AI活用が「管理職罰ゲーム」解消に繋がる可能性も指摘されています（日経ビジネス「AI活用は『管理職罰ゲーム』解消から」）。上司と部下のキャリア面談の効果を高める工夫や（パーソル総合研究所「上司と部下のキャリア面談の効果をより高めるには？」）、多様性時代におけるリーダー行動のアップデート、フォロワーシップの育成が注目されています（パーソル総合研究所「フォロワーシップはどう育むか」）。健全な対立を育み、風通しの良い組織を作るための対話の再設計も重要視されています（パーソル総合研究所「風通しの良い組織を作る4つのポイント」）。

人事制度・採用・育成

企業は人事制度の抜本的な刷新を進めており、カルビーが約17年ぶりに人事制度を刷新した事例が挙げられます（日本の人事部「カルビー、約17年ぶりに人事制度を抜本的に刷新」）。採用においては、候補者のエンゲージメントを最大化する「OfferLetterCX」の提供開始（リクレングス）や、「全社員で採用広報」を実現する動き（HRzine「『全社員で採用広報』を実現する方法」）が見られます。人的資本経営への意識が高まり（日本の人事部「人的資本経営に関する意識調査」）、ダイバーシティ経営推進に向けた具体的アクション実践事例集も公表されています（日本の人事部「ダイバーシティ経営推進に向けて企業に求められる具体的アクション実践事例集を公表」）。キャリア自律を促すための取り組みとして、NTTドコモのキャリア開発施策が紹介されており（パーソル総合研究所「NTTドコモが実践するキャリア開発施策」）、スキルベース組織やジョブ型人材マネジメントの「次の一手」が模索されています（【書籍紹介】『誰もが成長し活躍する会社のしくみ』、パーソル総合研究所「ジョブ型人材マネジメントの『次の一手』を考える」）。

HRテック・AI活用

HR領域におけるAI活用が本格化しており、「AI時代の転職・採用に関するアンケート調査」（日本の人事部）が実施されるなど、その動向が注目されています。採用プロセスでは、AIを活用したOB/OG訪問を実現する「タレントブック・キャンパス」（talentbook）や、企業と候補者の適合度合いを自動判定する「PeopleX AI面接」（PeopleX）、アバター型AI面接サービス「Our AI面接」（JetB）など、AI面接サービスが多様化しています。AI活用は個人の業務効率化に留まらず、組織全体の生産性向上へと広がる段階にあり（リクルートMS調べ）、人事業務向けの「Fusion Agentic Applications」（米オラクル）や、就労証明書発行業務の自動化（MIXI、オプロ「帳票DX for SmartHR」）など、業務のDXが進んでいます。一方で、生成AIの職場での普及には課題も指摘されており（パーソル総合研究所「生成AIはなぜ職場に広がらないのか」）、AIエージェントを構築・活用できる人材育成支援（ギブリー「Claude Cowork 活用支援」）も行われています。
