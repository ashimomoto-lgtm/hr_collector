2 HR界隈トレンド分析

1on1・マネジメント領域

1on1の機能不全が課題視されており、パーソル総合研究所のコラム「なぜ1on1は機能しないのか？」や「1on1という『旅』を始める」でその背景と再設計の必要性が議論されています。上司と部下の関係性やリーダーシップのあり方が問われており、若手社員の「ありのままを受け入れて偉いと褒めてほしい」という理想像（プレジデントオンライン）と、管理職が凡庸化する組織の病（ダイヤモンド・オンライン「あの人、管理職になってから変わったよね」）が指摘されています。組織の多様性を活かすためのリーダー行動のアップデートや、建設的なコンフリクトを育む重要性がパーソル総合研究所のコラムで考察され、心理的安全性やフォロワーシップの育成が注目されています。経営リーダーの育成や社員エンゲージメント向上（山田博之氏著「図解でわかる！ 社員エンゲージメントを高める7つの方法」）が重視される一方で、リスクばかりを指摘する「大課長」問題（林宏昌氏著「上司はリスクばかりを指摘する 会社を潰す『大課長』問題」）など、マネジメント層の課題も浮き彫りになっています。

人事制度・採用・育成

採用活動では、カジュアル面談の実態調査が行われ、ニトリが通年採用を導入するなど、多様な採用手法が広がっています。企業の人事制度改革も活発で、レオパレス21がキャリア形成支援の人事制度改定を実施し、カルビーが約17年ぶりに人事制度を刷新するなど、従業員のキャリア自律を促す動きが見られます。育成面では、Schoo for Businessが階層別の年間育成計画パッケージを提供開始するなど、体系的な社員育成への投資が進む一方で、「人材流失、人材獲得の困難による人材不足」が優先リスクの1位（デロイト調べ）とされています。ジョブ型人材マネジメント（NECビジネスインテリジェンスの事例）やサクセッションプランの策定・運用、そして「静かな退職」に関する調査など、多様な働き方やキャリア形成を支援する制度設計が求められています。また、約40年ぶりの物価上昇局面において、定昇よりベアを優先し「配分」の見える化を企業に促す提言（パーソル総合研究所）がなされるなど、報酬制度の見直しも進んでいます。

HRテック・AI活用

HR領域におけるAI活用が急速に進展しており、採用ではtalentbookがAI時代のOB/OG訪問を実現する「タレントブック・キャンパス」を開始し、マイナビとMQueはAI面接まで支援するサービス「WHAT」を提供しています。ギブリーのAI面接に候補者の回答を深掘りする「インタラクティブ」機能が追加されるなど、AIによる採用活動の高度化が進む一方で、HERPの調査では採用企業と就職・転職を行う人双方で「人が関わることへの意識は依然高い」とされています。人事システムでは、ビズリーチがオンボーディング支援サービス「Onboard AI」を提供し、On Technologiesは人事業務を自律的に行うAIエージェント「WorkOn」を開発するなど、AIを活用した業務効率化・自動化が加速しています。タレントマネジメントシステムの導入も進んでおり、兵庫県が「タレントパレット」を、一橋大学が「HRBrain タレントマネジメント」を導入するなど、データに基づいた人材管理の基盤が整備されています。リクルートMSの調査ではAI活用が「個人の業務効率化にとどまらず、組織全体の生産性向上へと広がる段階」にあると指摘され、三菱UFJリサーチ&コンサルティングからは「AI-Readyな企業の条件」としてメタデータとガバナンスの重要性が強調されています。
